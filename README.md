# CHGK Bot

Telegram-бот для игры в ЧГК в групповых чатах.

## Features

- `/start` запускает игру (опционально: `/start 1..10` для желаемой сложности).
- `/next` показывает ответ на текущий вопрос и задает следующий.
- `/stop` останавливает игру и показывает статистику сессии.
- Автораспознавание ответов по полям "Ответ" и "Зачет".
- Бота можно добавить в группу/супергруппу и играть командой.

## Как пользоваться

1. Добавьте бота в группу Telegram.
2. Выдайте боту право отправлять сообщения.
3. Запустите игру командой `/start`.
4. Для перехода к следующему вопросу используйте `/next`.
5. Для остановки игры используйте `/stop`.

## Тестирование

- В личном чате: проверьте, что `/start`, `/next`, `/stop` работают.
- В группе: проверьте ответы бота на команды и автофиксацию правильного ответа.
